Turais is a project IsardSAT is in the process of, which is developing a satellite-based method for surveying intertidal zones — the coastal areas that flood and drain with the tide. It's designed as an alternative to traditional hydrographic survey methods like MBES sonar and LiDAR, which require boats, equipment, and personnel in the field, and are constrained by weather, tide timing, and daylight.

Instead, Turais uses radar Earth Observation data from satellites such as Sentinel-1 and SWOT, combined with ISARDSAT's expertise in satellite altimetry and Earth observation built up over 20 years. The output is hydrographic survey data for intertidal regions that complies with International Hydrographic Organisation (IHO) standards.

The underlying data format is flexible, so the project's outputs can support different applications depending on what's needed — for example, tracking sediment movement to inform dredging operations, or monitoring marine protected areas.

Part of the plan for Turais is to use SWOT data as a guide for Sentinel 1 data. SWOT is in the unique position that it has height measurements in the easily accessible datasets. In this summer project, I will build a training set for a machine learning algorithm to use Sentinel 1 data in conjunction with SWOT data. Sentinel 1 data has a higher resolution than SWOT, so the idea is to train the model to get the best of both worlds between the two data sets.
